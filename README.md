# Проект на T3 Stack (Next.js, Prisma, NextAuth, TypeScript)

## Описание проекта
Данный проект является клиент-серверным веб-приложением для организации и проведения соревнований по картингу (чемпионата). Система позволяет следить за ходом чемпионата, получать информацию о этапах, управлять календарём чемпионата, этапами и заездами, публиковать результаты и формировать таблицу чемпионата.

Приложение предназначено для трёх категорий пользователей:
Зрители — просмотр ближайших и текущих этапов, просмотр календаря, результатов.
Пилоты — участие в этапах, возможность следить за результатами.
Судьи —  формирование результатов заездов, выдача штрафов.
Администратор - управление календарём, заездами, также имеет функционал зрителей, пилотов судей.

## Используемые технологии
**T3 Stack (Next.js, TypeScript, Tailwind, tRPC, Prisma, NextAuth)**
- Next.js – фреймворк для React: маршрутизация, SSR/SSG, API-роуты
- TypeScript – JavaScript с строгой типизацией
- Prisma – ORM для работы с базой данных
- NextAuth.js – аутентификация
- Tailwind CSS – CSS-фреймворк для стилизации

## Установка
1) Клонировать репозиторий

   ``git clone https://github.com/``
   
   ``cd ROMANOV_KURSOVOI``
3) Установить зависимости
   
   ``pnpm install``
5) Инициализация БД
   
   ``pnpm prisma migrate dev --name init``
7) Запуск контейнеров
   
   ``pnpm db:start``
9) Запуск разработки
    
   ``pnpm dev``
