# Проект на T3 Stack (Next.js, Prisma, NextAuth, TypeScript)

## Описание проекта
Данный проект является клиент-серверным веб-приложением проведение соревнований по смешенной системе. Система позволяет задать параметров турнира, добавить в турнир участников, проводить жеребьевки перед турами, ввод результатов каждого тура; вывод общей таблицы результатов.

Приложение предназначено для трёх категорий пользователей:
Пользователи(участники) — просмотр результатов турниров
Организатор - создание турниров, и их проведение
Администратор - имеет весь доступ к функционалу

## Используемые технологии
**T3 Stack (Next.js, TypeScript, Tailwind, tRPC, Prisma, NextAuth)**
- Next.js – фреймворк для React: маршрутизация, SSR/SSG, API-роуты
- TypeScript – JavaScript с строгой типизацией
- Prisma – ORM для работы с базой данных
- NextAuth.js – аутентификация
- Tailwind CSS – CSS-фреймворк для стилизации

## Установка
1) Клонировать репозиторий

   ``git clone https://github.com/``
   
   ``cd ROMANOV_KURSOVOI``
3) Установить зависимости
   
   ``pnpm install``
5) Инициализация БД
   
   ``pnpm prisma migrate dev --name init``
7) Запуск контейнеров
   
   ``pnpm db:start``
9) Запуск разработки
    
   ``pnpm dev``
